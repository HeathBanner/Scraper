# Scrapey Boi

Scrapey Boi is a news scraper. It currently pulls articles from CNN's US and World sections and stores it with in the database. User are then allowed to comment or favorite each article. I'm not too sure why they wouldn't just do all of that on the official site...but humans are weird, I suppose...
